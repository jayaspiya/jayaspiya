Retail Intelligence System

Research & Development

Harnessing the power of data to improve retail business through data analytics. The research was conducted using agile development methodology. The project uses various analytical & data management techniques like concurrency management, high availability, and security to improve the efficiency of the project. Oracle Linux 6.7 and Oracle Database 11g were setup for backend and Tableau for frontend data visualization. Furthermore, various concepts of data warehouse were implemented to improve the accessibility of data for analysis. Numerous ethical issues and physical disabilities like cognitive disabilities and visual impairment were also considered during the research and development of the project.

Everest Book Store

Android & Web Application

Everest Book has both android and web applications. NodeJs-MongoDB RestAPI backs the application backend. Vuejs Framework is used with custom CSS for the frontend web application and Kotlin for Android. The frontend & backend were served on Vercel and Heroku. MongoDB Atlas was used for data persistency and Cloudinary for CDN. The Web app has an Admin dashboard to edit the items and check the order list. A small WearOS application was also implemented for the android application and various sensor triggers were implemented on the android application.
